# Anonymizer

Локальный обезличиватель персональных данных (ПДн) в русскоязычном тексте.
На входе строка — на выходе та же строка, где ПДн заменены плоскими метками
(`[Имя]`, `[Телефон]`, `[Адрес]`, …). Цель — безопасно отправлять текст во внешние
облачные LLM, не передавая туда исходные данные.

Три принципа:

- **Необратимость** — замена односторонняя, таблицы соответствий нет, оригиналы
  не восстановить.
- **Полнота важнее точности** — пропуск ПДн хуже лишней маскировки; при сомнении
  система маскирует.
- **Строго локально** — всё детектирование выполняется на вашей машине;
  единственный сетевой вызов — к локальному серверу LM Studio (`localhost:1234`).

Детекция — ансамбль из четырёх независимых детекторов: regex (структурные ID),
[Natasha](https://github.com/natasha/natasha), [Presidio](https://microsoft.github.io/presidio/)
+ spaCy, и локальная LLM через LM Studio (семантический добор и финальная проверка).
Модель не зашита в код — подойдёт любая инструкт-модель со structured output и приличным
русским (требования и примеры моделей — в документации). Недоступный компонент молча
отключается — пайплайн работает на оставшихся.

## Запуск на Windows без кода

Для тех, кому нужен готовый сервис, а не библиотека — два bat-файла в корне репозитория. Нужен
только установленный [Python 3.10+](https://www.python.org/downloads/) (при установке отметьте
«Add Python to PATH»); VS Code не требуется.

1. Скачать проект: кнопка **Code → Download ZIP** на GitHub (или `git clone`), распаковать.
2. Двойной клик по **`install.bat`** — один раз: создаёт `.venv`, ставит зависимости, качает модель
   spaCy `ru_core_news_lg` (~0.5 ГБ) и генерирует API-ключ в `.env`. Первый запуск долгий.
3. Двойной клик по **`start.bat`** — поднимает HTTP API на `http://127.0.0.1:8077` (API-ключ
   показывается в окне; проверка — `/health`). Останов — Ctrl+C или закрыть окно.

Bat-файлы — **только для Windows**. На Linux/macOS ставьте вручную (см. документацию).

## Быстрый старт

```python
from anonymizer import anonymize

print(anonymize("Иванов Иван Иванович, тел. +7 916 123-45-67, ИНН 7707083893"))
# → "[Фамилия] [Имя] [Отчество], тел. [Телефон], ИНН [ИНН]"
```

Демо на встроенных примерах: `python -m anonymizer`.
HTTP API: `python run.py` (слушает `127.0.0.1:8077`, ключ — заголовок `X-API-Key`).
